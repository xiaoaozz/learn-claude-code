# Learn Claude Code - Web 启动指南

## 项目概述
这是一个 Next.js 静态导出网站，包含 Agent Loop 模拟器和 Showcase 展示专栏页面，支持多语言（中/英/日）。

## web - Next.js 前端应用

### 快速启动

```bash
cd web
npm run dev
```

**启动后访问**：http://localhost:3000

访问 Showcase 展示专栏：http://localhost:3000/zh/showcase

```yaml
subProjectPath: web
command: npm run dev
cwd: web
port: 3000
previewUrl: http://localhost:3000/zh/showcase
description: Next.js 前端应用，包含 Agent Loop 模拟器和 Showcase 展示专栏，支持中英日三语
```
